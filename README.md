## 2022年6月17日

### 初始化项目

Angular CLI version 8.3.0

Install ng-zorro-antd，minireset.css

导入预设全局样式：

- index.less：全局样式，引入子样式
- layout.less：class样式
- mixins.less：less的快捷函数
- varibles.less：预设颜色和字体的变量
- zorro.less：ng-zorro-antd的默认样式
